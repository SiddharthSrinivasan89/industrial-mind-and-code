# Industrial Mind & Code (IM&C)

**The factory mind, brought to life through code.**

A research initiative exploring what happens when cutting-edge AI
meets the hard, physical reality of manufacturing.

## Quick Start

```bash
# Create a new experiment
./imc-new-experiment.sh experiment-name

# Run an experiment
cd experiments/imc-XX-name
python src/main.py
```

## Structure

- `shared/` - Shared utilities (Azure client, tracker)
- `templates/` - Experiment template
- `experiments/` - All experiments (each gets its own GitHub repo)
- `docs/` - Setup guide, roadmap
