# IM&C Experiment 01: Agentic Bullwhip Effect

## Research Question
How do LLM-based ordering agents respond to contextual information about Indian automotive supply chains, and does this contextual awareness reduce order volatility (bullwhip effect)?

## Hypothesis
Agents with richer context (market trends, supplier constraints, cultural factors like festivals) will make more stable ordering decisions compared to agents with minimal context.

## Methodology
- **Agent Design**: Azure OpenAI-powered ordering agents with varying context levels
- **Approach**: Behavioral observation (not forecasting accuracy)
- **Supply Chain**: Indian automotive parts suppliers
- **Variables**: Context richness, demand patterns, cultural events (Diwali, festival seasons)
- **Metrics**: Order volatility, variance amplification, decision stability

## Expected Outcomes
1. Quantify how context affects ordering behavior in LLM agents
2. Identify which contextual factors most reduce bullwhip effect
3. Establish baseline for agentic system behavior in supply chains
